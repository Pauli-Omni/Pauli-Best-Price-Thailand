# Master-Aufnahme für AudioSegmentService (Flutter) und osg_audio_segment_service.js (Web).

Lege hier `search_intro_long.mp3` ab — eine durchgehende Aufnahme (~14 s) mit drei Blöcken:

| Segment | Zeit | Inhalt (Referenz EN) |
|---------|------|----------------------|
| welcome_short | 0–4 s | „Hello! I'm Pauli.“ |
| search_action | 4–8 s | „Let me search the best prices.“ |
| save_money | 8–14 s | „I will help you save money.“ |

Web-Konfiguration: `public/sounds/pauli/audio-segments.json` (Timings anpassen).

Ohne diese Datei: Fallback auf einzelne MP3s unter `public/sounds/pauli/th/`.
